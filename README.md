# barrage-website
这是一个弹幕留言网站


## 使用框架介绍
+ 前端：
 + 原生**javascript**,**html5**,**css3**,以及**jQuery**

+ 后端:
 + **springboot2.0** **mysql** **JpaRepository** **freemarker** 

+ 存在的问题:
 + 功能单一
 + 暂时只支持pc端
 + 手机端某些浏览器无法实现全部功能
