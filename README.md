# barrage-website
这是一个弹幕留言网站，提供一些自己觉得有意思的小东西。


## 使用框架介绍
+ 前端：
 + 原生**javascript**,**html5**,**css3**,以及**jQuery**,**bootstrap**

+ 后端:
 + **springboot2.0** **mysql** **JpaRepository** **freemarker** **Mybatis** **Kafka** **Redis**

## 设备支持情况
+  手机端支持（横屏）
+  pc端完全支持(搜狗浏览器,谷歌浏览器,IE浏览器)
## 存在的问题:
 + 手机端请横屏使用
